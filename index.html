  • git clone path-to-repository - вставляем путь к репозиторию
  • git status - статус текущего локального репозитория
  • git add index.html - добавить новый файл под контроль ГИТА
  • git commit -am"added new file index.html - зафиксировать изменения и дать им комментарии
  • git push - отправить все закоммиченные измения на репозиторий на сервер
  • git pull - забрать свежие обновления
